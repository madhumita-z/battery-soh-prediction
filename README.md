# Battery SOH Prediction

A machine learning project for estimating lithium-ion battery State of Health (SOH) using battery operating characteristics.

## Features

* Battery SOH Prediction
* CSV Upload Analysis
* Advanced Manual Input
* Quick Health Check
* Streamlit Web Application

## Technologies Used

* Python
* Pandas
* Scikit-Learn
* Streamlit

## Machine Learning Models

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

## Project Workflow

Battery Data → Feature Extraction → Model Training → SOH Prediction → Health Assessment
